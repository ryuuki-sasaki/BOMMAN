★説明
・爆弾を使って戦うゲーム
・一定時間経過でフィールドが狭まる(その際配置される壁と同じ位置にいるとキャラクターが倒れる)
・2P対戦とCPU対戦
・オプション(タイトル画面で選択)
	・当たると倒れてしまう雷
	・当たるとアイテムが散ってしまう台風
各アイテム(画面左から表示順)
・ノーマルボム：通常ボム
・プラスボム：一度にプラスボムアイテムの数だけ通常ボムを置ける
・大爆発ボム：ノーマル、プラスボムの爆発範囲が広くなる
・痺れボム：キャラクターの動きを停止
・ボムを投げる：狙った場所にボムを投げる

★アピールポイント
	CPUの動作。
		・爆発範囲から逃れる所。
		・爆発範囲から逃れた後、停止して待つところ。
		・停止中に爆弾を置かれると非難するところ。
		・周りの状況やキャラクターごとに応じて、壁を破壊するかキャラクターに対して爆弾を置くかの優先順位を判断する。
